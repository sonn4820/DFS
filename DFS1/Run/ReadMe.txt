How to use:

F1: DEBUG DRAW / F2: DEBUG MODE
K/L to change music
_______________________________________
Game Mode

WASD to move
Space to jump
1-2-3 to select option
Mouse wheel to change value
LMB to shoot (hold/release)
RMB to Dribble 
F to pick up ball
N to spawn ball
_______________________________________
Debug Mode

WASD to move
Q/E to change option
Mouse wheel to change value
LMB to shoot
C to clear all balls
N / M to spawn ball
